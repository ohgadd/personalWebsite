Welcome

Hi, I'm Josh Gadd and my passion lies within front-end web development.

I first began web development during my college course, where I learned about basic HTML4 and CSS. This included things such as the box model and basic styling. It was not until a while later that I realised this is what I wanted to do. During my university course, which was based on Computer Forensics and Security, the first year held a unit based on User Centered Web Development which is where I started to notice I was actually enjoying completing assignments. Later on, in second year I took an optional unit based on Web Technology Integration, in which we learned about Content Management Systems and this is where I really found out what I wanted to pursue, which was front-end web development.


Key Skills

HTML5

I began with HTML4 during college, as HTML5 was only just beginning to appear. HTML5 is now the upper standard of basic web-based languages due to its much improved handing of web applications, pretty much destroying any use of Adobe Flash.

CSS3

I began learning about CSS at the exact same time as HTML4 in college, and quickly learned it was a necessity to have alongside HTML inorder to have a website that not only performs well, but looks good. CSS3 is still getting regularly updated as it is constantly undergoing development.

Bootstrap 4

Bootstrap is pretty much an open-source 'styling package' that was 'designed for everyone, everywhere' and is known for its great responsive design. It contains parts of HTML, CSS and JS framework in order to make designing responsive pages much faster and more efficient.

University Wordpress Website, for a 'Client'

We also had to ensure the website was responsive depending on device, which was my first experience at building a responsive website.
